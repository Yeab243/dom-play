Captain Phillips Rehearsal Tool

  • This is a webpage that helps the actors of "Captain Phillips" touch the name of the character they are playing so it highlights their lines to make it easier to rehearse.
  • Created a small, single page application using the DOM.
  • With a single click, it highlights all lines for a specific actor in a play yellow and the rest grey.
  • The page performs this action without page re-loads.
  • Created a function named highlight which passes in the span element by reference, meaning the live instance of the element on the page, and changes the background color.
  • Added the capability of toggling (switching between 2 states) the background color of the element passed into the function.
  • To wire events to a set of elements, placed the actor spans into a div with an id of play.
  • Wired a simple function to each of the spans inside the div.
  • The advantage is using very little code to add events and functions to each item all at once.

Click the Captain Phillips navigation button, at the top, to review the rehearsal.
